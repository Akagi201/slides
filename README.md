## Install tools

* `npm install -g @marp-team/marp-cli`
* vscode marp plugin

## Preview Local

```sh
marp -p -s ./markdown
```

## Export to PDF

```sh
marp --pdf --allow-local-files ./markdown/xxx.md -o ./pdf/xxx.pdf
```
